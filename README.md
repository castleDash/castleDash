
#Castle-Dash

A two dimensional adventure game that focuses on player momentum to traverse increasingly difficult levels and defeating enemies.

##Technology Used

 * JavaScript
 * Java
 * H2
 * Spring/Hibernate
 * phaser.io
 * Cocoon (roadmap)
